Installationsanleitung für modified Shop

Dateien kopieren

  1. Kopieren Sie die Dateien aus dem Verzeichnis shop in das Hauptverzeichnis Ihres Shops.
  2. Kopieren Sie die Dateien aus dem Verzeichnis admin in das Admin-Verzeichnis Ihres Shops.

Modul aktivieren

  1. Shop Admin öffnen
  2. Im Menü unter "Module" die Seite "System Module" öffnen
  3. In der Liste das Modul "releva.nz" anwählen
  4. Auf die Schaltfläche "Installieren" klicken
  5. Das Modul befindet sich in der Navigation unter "Partner Module"

Modul einrichten

  1. Klicken Sie in der Navigation auf "Partner Module", dann auf "releva.nz" und anschließend auf "releva.nz - Konfiguration".
  2. Hinterlegen Sie im entsprechenden Eingabefeld den von uns bereitgestellten API-Key.
  3. Klicken Sie auf die Schaltfläche "Speichern".
  4. Wenn der API-Key korrekt ist, wird Ihnen eine entsprechende Erfolgsmeldung angezeigt. Falls nicht, überprüfen Sie bitte, ob Sie den API-Key korrekt eingegeben haben.
  5. Die Einrichtung ist nun erfolgreich abgeschloßen. Sie haben nun Zugriff auf Statistiken und die Konfiguration ihres Werbebudgets unter dem Menüpunkt "Partner Module" → "releva.nz" → "releva.nz - Statistik".
